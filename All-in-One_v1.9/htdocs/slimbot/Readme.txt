Zum Bot:
Zuerst musst du den SQL Code der Datei bot.sql ausführen, das geht so:
- http://localhost/phpmyadmin/ im Browser eingeben
- Dann die Datenbank Lan oder DSLan, jenachdem, auswählen
- Dann oben auf "SQL" und dann den Code aus bot.sql einfügen und auf OK klicken
- Dann http://localhost/slimbot/
- Hier kannst du den Bot dann starten und die Config ändern

PS: Es hat schon seinen Sinn, dass die config.php verschlüsselt ist... 
Nicht, dass du denkst...
Die Config änderst du dann wie gesagt beim Bot selber...
